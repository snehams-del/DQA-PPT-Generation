Get Started

1

Install agents-cli

pip install agents-cli
2

Clone this repo

git clone https://github.com/your-org/vertical-agent-skills
cd vertical-agent-skills
3

Open a skill in your agent

# List available skills
./vs list

# Open with Gemini CLI or Claude Code — the agent walks you through setup
./vs gemini retail-product-search
./vs claude retail-virtual-tryon
4

Evaluate & improve

# Run evaluations
./vs eval retail-virtual-tryon --project-id $PROJECT

# Run self-improving optimization loop
./vs improve retail-virtual-tryon --rounds 5 --project-id $PROJECT
