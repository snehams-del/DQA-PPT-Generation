# Model Garden Deploy Agent
An ADK-based agent that provides a chat interface for using GenAI models with Vertex AI Model Garden.

## Installation
Follow the instructions at https://google.github.io/adk-docs/ to install ADK and set up your `.env` file.

## Usage

**Web Mode**: Run the agent in web mode with this command:

```bash
cd adk-samples/python/agents/
adk web model_garden_agent
```

**Command-line Mode**: Run the agent in command-line interactive mode with this command:

```bash
cd adk-samples/python/agents/model_garden_agent
adk run model_garden_agent
```
